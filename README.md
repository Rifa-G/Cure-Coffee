### Installation & Development

1. **Install dependencies:**
   ```bash
   bun install
   ```

2. **Start development server:**
   ```bash
   bun run dev
   ```
   The website will be available at `http://localhost:5173`

3. **Build for production:**
   ```bash
   bun run build
   ```

4. **Preview production build:**
   ```bash
   bun run preview
   ```

5. **Lint code:**
   ```bash
   bun lint
   ```
